## Interpretation and maintenance

- The World Bank Atlas link is a provenance lead, not a blanket license for every embedded dataset.
- `mixed-and-partly-unknown` means only the specifically evidenced component is identified; it does not cover neighboring files.
- `inherits-unknown-input-rights` means a generated output remains subject to unresolved rights in its inputs.
- Add a collection-specific source revision, retrieval date, license evidence, sensitivity review, and SHA-256 manifest at the next verified refresh.
- Large-file rules and the non-destructive LFS/DVC migration plan are in `docs/STORAGE_POLICY.md`.
- Validate this catalog with `python3 scripts/check-data-governance.py`.
