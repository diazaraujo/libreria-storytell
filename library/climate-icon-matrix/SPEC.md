# climate-icon-matrix

**Atlas:** `sdg13_icon_matrix`  
**Narrative:** Each cell ≈ 8 million people; rows aggregate regional shares of vulnerability / high risk.

## Data

- `region_code`, `sha_vuln`, `sha_exp`, `sha_risk`, `totalpop`, `pop_vuln`, `pop_risk`
